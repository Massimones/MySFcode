The 4 examples (Cuboid, Parall, Spherical_shell and General_shape,
show the effects of TPE inclusions with different geometries.

If your PC is running slowly, please reduce the number of the observation
points in the input files.

"plot_results_MAP.m" is a Matlab script which can be used to plot the
resulting displacement, stress and strain fields.

The distributions of single forces employed in the "efcmp.inp" files
were obtained with the MATLAB scripts contained 
in the ../Generate_SF_distribution folders
